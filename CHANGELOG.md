# Changelog

All notable changes to the `investment-strategy-cskill` will be documented in this file.

## [1.0.0] - 2025-01-04

### Added
- **Core Modes**: Implementation of Daily Check-in, Weekly Review, and Monthly Strategy workflows.
- **Data Fetchers**: Live price integration for Yahoo Finance (IDX/US) and CoinGecko (Crypto).
- **Calculators**: Automated Kelly Criterion, Sharpe Ratio, Sortino Ratio, and Max Drawdown.
- **Trackers**: Portfolio manager, Trade Journal (JSON), and Emergency Fund progress tracker.
- **Phase Detection**: Automatic assessment of Phase 1 (Foundation) and Phase 2 (Wealth Accumulation).
- **Initialization**: `init portfolio` command to set up templates and sample data.
- **Documentation**: Comprehensive SKILL.md and reference guides for all methodologies.

### Changed
- Initial project structure created.
- Modernized dependency management using `uv`.
