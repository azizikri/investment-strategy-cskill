# Daily Check-In Checklist

## Morning Routine (2-3 minutes)

### Portfolio Review
- [ ] Check total portfolio value
- [ ] Review 24h price changes
- [ ] Note any significant moves (>5%)

### Market Awareness
- [ ] Check market status (open/closed)
- [ ] Note any major news affecting holdings
- [ ] Review forex rates (USD/IDR)

### Alerts Check
- [ ] Allocation drift alerts
- [ ] Emergency Fund progress
- [ ] Phase status (1 or 2)

### Mental State
- [ ] Am I emotionally stable today?
- [ ] Any urge to make impulsive trades?
- [ ] Sticking to the investment plan?

## Quick Actions
- **If market closed**: Focus on research, no trading
- **If allocation drifted >5%**: Note for weekly review
- **If news-driven volatility**: No panic selling
